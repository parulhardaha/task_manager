## Pproject Description :
This is a full-stack Team Task Manager web application built using Flask.
It allows users to create and manage projects, assign tasks to team members, and track their progress.
The application includes authentication (signup/login), role-based access (Admin and Member), and a dashboard to monitor task status and deadlines.
Admins can add team members and manage projects, while members can view and update their assigned tasks. 
The system is designed to improve team collaboration and task organization.
